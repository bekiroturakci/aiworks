Başlangıç

Claude Code'u kurun (Node.js 18+ gereklidir):

# Kurulum (yöntem 1 — resmi)
curl -fsSL https://claude.ai/install.sh | bash

# Kurulum (yöntem 2 — npm ile)
npm install -g @anthropic-ai/claude-code

# Projeye gidin ve başlatın
cd proje-klasoru
claude
/init
✓ /init komutu kod tabanınızı tarar ve bir başlangıç CLAUDE.md dosyası oluşturur.

Hook'ları Kurma

Hook'lar = Deterministik geri çağırmalar. Claude bir araç kullanmadan önce/sonra otomatik çalışır.

PreToolUse PostToolUse Notification Stop SessionStart UserPromptSubmit SubagentStart ve daha fazlası... 
{
  "hooks": {
    "PreToolUse": [
      {
        "matcher": "Bash",
        "hooks": [
          {
            "type": "command",
            "command": "scripts/sec.sh",
            "timeout": 5
          }
        ]
      }
    ]
  }
}
0 → İzin ver (devam et)
2 → Engelle (durdur)
i Stop hook'u, Claude yanıtını tamamladığında tetiklenir — CI/CD entegrasyonları için idealdir.
9
İzinler ve Güvenlik
{
  "permissions": {
    "allow": [
      "Read:*",
      "Bash:git:*",
      "Write:*:*.md"
    ],
    "deny": [
      "Read:env:*",
      "Bash:sudo:*"
    ]
  }
}
  • allow — Claude'un izinsiz kullanabileceği araçlar
  • deny — Her zaman engellenen araçlar
  • settings.json takım ile paylaşılır (Git)
  • settings.local.json kişiseldir (.gitignore)
! deny kuralları allow kurallarını her zaman geçersiz kılar. Güvenlik önceliklidir.

MCP Sunucuları EK

Model Context Protocol (MCP) sunucuları, Claude Code'a harici araçlar ve veri kaynakları bağlamanızı sağlar. 

// settings.json içinde MCP yapılandırması
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y",
        "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_TOKEN": "..."
      }
    }
  }
}
  • GitHub, Slack, veritabanları gibi harici servislere bağlanın
  • Özel MCP sunucuları yazarak Claude'un yeteneklerini genişletin
  • settings.json veya settings.local.json içinde tanımlayın
+
CI/CD Entegrasyonu EK

Claude Code, başsız (headless) modda CI/CD pipeline'larında kullanılabilir. 

# Etkileşimsiz mod (pipe ile)
echo "Bu PR'ı incele" | claude -p

# JSON çıktı formatı
echo "Testleri çalıştır" | claude -p \
  --output-format json

# Maksimum tur sayısı belirleme
echo "Hataları düzelt" | claude -p \
  --max-turns 10
✓ -p (print) bayrağı, Claude'u etkileşimsiz modda çalıştırır. GitHub Actions, GitLab CI vb. ile entegre edin.
